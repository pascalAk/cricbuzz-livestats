# Cricbuzz LiveStats

Real-Time Cricket Insights and SQL-Based Analytics

## Tech Stack
Python, Streamlit, MySQL, Cricbuzz API, Pandas, Power BI

## Features
- Live match scores and scorecards from Cricbuzz API
- Real-time batting and bowling rankings
- 25 SQL queries from beginner to advanced level
- Full CRUD operations on player data
- Custom SQL query interface
- Power BI dashboard connected to MySQL

## Database
- 7 tables, 1175 real IPL matches, 786 players, 59 venues
- Data sourced from Cricsheet ball-by-ball JSON files

## Setup
1. Clone the repository
2. Create virtual environment and install requirements
3. Configure .env with MySQL credentials and RapidAPI key
4. Run schema.sql and seed_data.sql in MySQL Workbench
5. Run the app
